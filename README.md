# CodePet 🐾

CodePet is an interactive and enjoyable, Java-based Quiz Management System designed for teachers and students.  
It provides an easy way to create, manage, attempt, and evaluate quizzes using a simple and user-friendly interface built with Java Swing.

---

## 📌 Features

### 👨‍🏫 Teacher Panel
- Add new quiz questions
- View all existing questions
- Delete unwanted questions
- Manage quiz content easily

### 👨‍🎓 Student Panel
- Attempt quizzes
- Submit answers
- View score after quiz completion

### 🔐 Login System
- Separate login for teachers and students
- Secure access to respective dashboards

### 💾 Database Integration
- MySQL database connectivity using JDBC
- Stores questions, answers, and user data efficiently

### 🖥️ GUI Interface
- Built using Java Swing
- Clean and simple desktop application interface

---

## 🛠️ Technologies Used

- **Java**
- **Java Swing**
- **MySQL**
- **JDBC**
- **IntelliJ IDEA**
